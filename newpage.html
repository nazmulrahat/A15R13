Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML ব্যবহার করা হয় যখন HTML কনটেন্ট প্রয়োজন।
2. innerText ব্যবহার করা হয় শুধুমাত্র দৃশ্যমান টেক্সট পেতে।
3. textContent ব্যবহার করা হয় দ্রুততর ও সম্পূর্ণ টেক্সট পেতে ভালো।

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where an event listener is added to a parent element, which can manage events for its child elements.

Question-4: What is event bubbling in the DOM?

Event Bubbling is when an event starts at the target element and bubbles up to its parent elements, up to the root.

Question-5: How do you create, add, and remove elements using JavaScript?

I can create, add, and remove elements using JavaScript like this :
1. Create: document.createElement('tagName')
2. Add: parentElement.appendChild(newElement)
3. Remove: element.remove() or parent.removeChild(childElement)
